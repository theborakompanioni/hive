chesshive
==================================================

#### Prerequisites

* Node JS
* Mongo DB up and running on the default port (27017) and using the default database (test)
* ElasticSearch server up and running on the default port (9200)

#### Run the application

```
$> npm install
$> bower install
$> node initData.js
$> node .
```

Browse the following address: `http://localhost:3000`
