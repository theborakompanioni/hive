chesshive
==================================================

#### Run the application

Build the UI
```
cd client/apps/hivechess
$> npm install
$> bower install
$> export NODE_ENV=development && grunt
```

Run
```
$> npm install
$> export NODE_ENV=development && node .
```

Browse the following address: `http://localhost:3000`
