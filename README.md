chesshive
==================================================

## Run the application

### Build the UI
```
cd client/apps/chesshive
$> npm install
$> bower install
$> export NODE_ENV=development && grunt
```

### Run
```
$> npm install
```

##### Development
```
$> export NODE_ENV=development && node .
```
##### "Production"
```
$> export NODE_ENV=production && node .
```

Browse the following address: `http://localhost:3000`
